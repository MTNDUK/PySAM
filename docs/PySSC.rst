PySSC
*****

Generic wrapper for SAM Simulation Core models using `sscapi.h <https://github.com/NREL/ssc/blob/develop/ssc/sscapi.h>`_

This original Python SDK, :doc:`PySSC`, is bundled, and can be imported by::

	from PySAM.PySSC import PySSC
